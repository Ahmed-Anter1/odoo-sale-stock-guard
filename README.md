# Odoo Sale & Stock Guard

Installable Odoo 18 addon that validates warehouse availability before confirming a quotation and preparing its invoice.

## Features

- Location-aware free-quantity validation
- Dedicated stock-check action
- Controlled quotation confirmation
- Delivery reservation
- Draft invoice creation
- Clear shortage messages

Module: `portfolio_sale_guard`  
Dependencies: `sale_stock`, `account`
